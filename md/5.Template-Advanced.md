# Decorator in C++

using template to decorate a funciton

函数包装器用途    

1. 设计执行接口,接口设计关卡（），计数
2. 函数包装器依赖于函数模板，实现通用泛型
3. 函数代码可以内嵌在另外一个函数，实现函数怀孕
4. 函数包装器，用于管理内嵌函数，外部函数调用

```c++
//函数包装器， T数据类型， F是函数
template<typename T, typename F>
T test(T t, F f)
{
    static int count = 0;
    count++;//计数器
    std::cout<<"tempate function called "<< count << "times"<<std::endl;
    return f(t);   
}
```
