# 函数指针的引用

函数指针
```cpp
    int (*p)(int, int);//指针指向函数
    p = jia;
```

函数指针的引用

```cpp
    int (* & rp)(int, int)(jia); //必须初始化
```

如果函数返回值是一个函数指针的引用如何表达：
```cpp
    int (*& )(int, int)；//先分解数据类型，再考虑引用
    int (*& chp(int(*& rp)(int, int)))(int, int) //int(*& rp)(int, int))作为参数
    {
	rp = jia;
	return rp;
    }
    
    
```


```c++
void change(int(* & rp)(int,int))
{
    rp = jia;
}
```
